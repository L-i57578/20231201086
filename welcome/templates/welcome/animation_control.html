JavaScript动画控制功能演示

这个页面展示了如何使用JavaScript控制CSS动画的播放和暂停。

CSS动画定义代码：

@keyframes move {
    0% {
        left: 0%;
    }
    50% {
        left: 50%;
    }
    100% {
        left: 0%;
    }
}

JavaScript动画控制代码：

document.addEventListener('DOMContentLoaded', function() {
    // Find heading
    const h1 = document.querySelector('h1');

    // Pause Animation by default
    h1.style.animationPlayState = 'paused';

    // Wait for button to be clicked
    document.querySelector('button').onclick = () => {

        // If animation is currently paused, begin playing it
        if (h1.style.animationPlayState == 'paused') {
            h1.style.animationPlayState = 'running';
        }

        // Otherwise, pause the animation
        else {
            h1.style.animationPlayState = 'paused';
        }
    }
})

JavaScript代码说明：

  • DOMContentLoaded事件 - 确保页面完全加载后再执行JavaScript代码
  • document.querySelector('h1') - 获取页面中的h1元素
  • animationPlayState属性 - 控制CSS动画的播放状态（'running'或'paused'）
  • onclick事件处理 - 当按钮被点击时切换动画状态
  • 条件判断 - 根据当前动画状态决定是播放还是暂停

功能特点：

  • 页面加载时动画默认暂停
  • 点击按钮可以切换动画的播放/暂停状态
  • 按钮有视觉反馈（状态指示器和文字提示）
  • 动画状态实时显示
  • 支持无限循环动画的控制
