CSS动画演示

这个页面展示了CSS @keyframes动画的基本用法。

动画定义代码：

@keyframes grow {
    from {
        font-size: 20px;
    }
    to {
        font-size: 100px;
    }
}

动画应用代码：

h1 {
    animation-name: grow;
    animation-duration: 2s;
    animation-fill-mode: forwards;
}

动画属性说明：

  • animation-name: 指定要使用的@keyframes动画名称
  • animation-duration: 设置动画持续时间为2秒
  • animation-fill-mode: forwards 动画结束后保持最后一帧的状态
