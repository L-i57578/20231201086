CSS位置移动动画演示

这个页面展示了CSS @keyframes动画实现元素位置移动的效果。

动画定义代码：

@keyframes move {
    from {
        left: 0%;
    }
    to {
        left: 50%;
    }
}

动画应用代码：

h1 {
    position: relative;
    animation-name: move;
    animation-duration: 2s;
    animation-fill-mode: forwards;
}

动画属性说明：

  • position: relative - 设置相对定位，使left属性生效
  • animation-name: move - 指定使用名为"move"的@keyframes动画
  • animation-duration: 2s - 设置动画持续时间为2秒
  • animation-fill-mode: forwards - 动画结束后保持最后一帧的状态

动画效果：

"Welcome!"文字会从屏幕左侧(0%)水平移动到屏幕中央(50%)的位置。
